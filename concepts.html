External Stylesheets & Font Embedding

I used an external CSS file and a custom font.

@font-face {
font-family: 'Tranformers';
src: url("Fonts/Transformers Movie.ttf") format('truetype');
}

Why This Works:

  • @font-face lets you load a custom dont that isn't installed on the users computer
  • This makes your site look unique and consistent on every device.
  • Applies to titles and tables

Reusable Global CSS (Site-Wide Styling)

Created universal styles that apply to every page.

body {
color: white;
margin: 0;

Why This Works:

  • If the footer or navbar is changed once, it changes on every page .
  • This represents a "Don't repeat youself" design

Flexbox Layout

Used Flexbox to align text and images side by side.

.DecepticonLayout {
display: flex;
gap: 30px;
align-items: flex-start;
}

Why This Works:

  • display:flex turns the container into a flexible layout.
  • gap creates spacing between columns.
  • flex properties control column width.

Responsive Background Images

Each page uses full-screen background images.

.decepticonsImage {
min-height: 100vh;
background-image:url(...);
background-size: cover;
background-attachment: fixed;
}

Why This Works:

  • 100vh makes the page fill the entire screen height.
  • background-size: cover scales the image to fill the screen.
  • background-attachment: fixed creates a parallex effect when scrolling.
